# Contributing

If you are not seeing a given asset or procedure and would like to contribute it to the
repository, please follow these guidelines:

* Open an issue describing the asset or procedure you would like to contribute.
* Fork the repository and create a new branch for your contribution.
* Add your asset or procedure to the appropriate directory in the repository.
* Ensure that your contribution passes the `cargo run` command without errors.
* Submit a pull request describing your contribution and referencing the issue you opened.
